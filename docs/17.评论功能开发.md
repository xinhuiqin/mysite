# 创建评论应用comment
  - 创建应用
    ``` 
    (venv) D:\>cd mysite
    (venv) D:\mysite>cd apps
    (venv) D:\mysite\apps>python3 ../manage.py startapp comment
    ```
  - 注册应用
    <br/>mysite/mysite/settings.py
    ``` 
    INSTALLED_APPS = [
       'comment.apps.CommentConfig',
    ]
    ```
  - 应用设置
  <br/>mysite/apps/home/apps.py
  ``` 
  class HomeConfig(AppConfig):
    name = 'home'
    verbose_name = '评论' 
  ```
