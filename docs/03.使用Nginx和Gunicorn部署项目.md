# 创建用户(可选)并切换到新用户
一般我们都是以root用户身份登陆服务器，但是以root用户身份部署代码不安全，所以需要创建一个新用户。
然后以新用户的身份部署项目。
``` 
# 创建名为Sweeneys的新用户
$ useradd -m -s /bin/bash Sweeneys

# 把新创建的用户Sweeneys加入超级权限组
$ usermod -a -G sudo Sweeneys

# 为新用户设置密码。注意:在输密码的时候不会有字符显示
$ passwd Sweeneys

# 切换到创建的新用户
$ su - Sweeneys
```
